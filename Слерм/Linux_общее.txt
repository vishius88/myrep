Чта такое поперационная система:
Операционная система — это программный комплекс, который управляет аппаратными ресурсами компьютера и предоставляет приложениям и пользователю удобный интерфейс для работы с ними.
Она обеспечивает запуск программ, управление памятью, процессами, устройствами ввода-вывода и файловой системой.


Как происходит загрузка ОС:
После включения сервера BIOS/UEFI выполняет аппаратную проверку и передаёт управление загрузчику (например, GRUB). Загрузчик загружает ядро Linux и initramfs, ядро инициализирует драйверы и монтирует корневую файловую систему. Затем запускается первый процесс init (чаще всего systemd), который монтирует остальные файловые системы, стартует сервисы и демоны. В конце запускается getty, который показывает приглашение login: и ждёт ввода пользователя.

Питание → BIOS/UEFI → Загрузчик → Ядро → initramfs → root FS → init (systemd) → сервисы → getty → login prompt
### Развернуто
Подача питания и аппаратная инициализация

PSU подает питание → материнка инициализирует CPU, RAM, чипсет.

BIOS/UEFI выполняет POST (Power-On Self Test).

Выбор устройства загрузки

BIOS/UEFI читает таблицу приоритетов загрузки → находит загрузочное устройство (HDD, SSD, NVMe, PXE).

Для UEFI: загружается .efi файл из EFI System Partition (ESP).

Для BIOS: считывается MBR (первые 512 байт).

Загрузчик (GRUB, systemd-boot, LILO и т.д.)

Загружает в память ядро (vmlinuz) и initramfs/initrd.

Передает управление ядру, передавая параметры командной строки (kernel parameters).

Ядро Linux

Распаковывается в память.

Инициализирует драйверы (основные — из initramfs).

Монтирует временный root из initramfs.

Запускает процесс init (PID 1).

Init система (systemd, SysVinit, OpenRC и т.д.)

Монтирует реальный root (из /etc/fstab).

Запускает сервисы, таймеры, сокеты, цели (targets).

Настраивает сеть, логи, демоны.

Подготовка к интерактивной работе

Запускается getty (или agetty) для TTY.

Появляется приглашение для входа (login prompt).
